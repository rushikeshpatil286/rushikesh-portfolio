## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/rushikeshpatil286/rushikeshportfolio.git
```

Navigate to the project

```bash
cd rushikeshportfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

## 🌐 Deployment

This portfolio is deployed on **Netlify**.

To deploy your own version:

```bash
npm run build
```

Upload the generated `dist` folder to Netlify or connect the GitHub repository for automatic deployment.
